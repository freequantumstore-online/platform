# AGENTNAME — Classifier Agent

A **classifier** heuristic agent on [FreeQuantumStore](https://freequantumstore.online).

Classifiers answer yes/no questions about input data with a confidence score:
"Is this a login page?", "Is this spam?", "Is this a job listing?", "Is the user logged in?"

## How classifiers work

1. **Signals**: Define patterns to look for (structural, textual, semantic)
2. **Weights**: Each signal has a positive or negative weight
3. **Score**: Sum weighted signals, normalize to 0-1 confidence
4. **Threshold**: Confidence >= threshold = positive classification

The heuristic code is generated by an LLM and evolved against test fixtures.
At runtime: zero model, zero tokens, sub-millisecond classification.

## Output shape

Every classifier returns:
```ts
{ match: boolean, confidence: number, signals: Signal[] }
```

## Training

1. Add test cases to `web/src/heuristic.test.ts`
2. Use the [Trainer](https://freequantumstore.online/a/trainer/) to evolve the heuristic
3. Or write the heuristic by hand — the pattern is simple signal scoring

## Use cases

- Login page detection, job listing detection, auth state detection
- Error/404/500 page detection, cookie banner detection, captcha detection
- Paywall detection, bot-check detection, GDPR consent detection

Each saves ~500-1000 LLM tokens per page vs sending HTML to an API.
